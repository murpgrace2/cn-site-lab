# cn-site-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为通用 HTML 页面资料的存储与展示空间。

## 项目简介

`cn-site-lab` 收集并维护一系列独立的 HTML 文件，这些文件可能包含静态信息页、演示页、工具页等。  
每个页面均为自包含的 HTML 文件，不依赖外部资源（或依赖已声明的外部资源），便于直接查看或托管。

## 目录说明

```
.
├── index.html          # 仓库主页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：每个 HTML 文件为一个独立页面，命名清晰，便于识别内容。
- **assets/**：若多个页面共享资源，可放置于此。推荐各页面尽量内嵌样式与脚本，保持高度独立。

## 页面归档说明

本仓库仅负责归档与展示，不保证所有页面长期可用或内容更新。  
贡献者可以提交新的 HTML 页面，或对现有页面进行修正，但需遵循以下原则：

- 每个页面应为完整、可独立运行的 HTML 文档。
- 页面内容应不含恶意代码、跟踪脚本或违反法律法规的信息。
- 如有外部依赖（如 CDN 库），请在页面内明确标注来源。

## 维护说明

- 本仓库由社区维护，欢迎通过 Pull Request 提交新页面或改进。
- 对于不再适用或存在问题的页面，仓库维护者有权移除或修改。
- 提交前请确保页面在主流浏览器中基本可用，并简要说明页面用途。
